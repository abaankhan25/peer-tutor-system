# Peer Tutor System

## Description  
A web-based platform where students can connect with peer tutors for academic help.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP (Backend - planned)
- MySQL (Database - planned)

## Features
- User Registration & Login
- Tutor Request System
- Interactive UI
- Responsive Design

## How to Run
1. Download the project
2. Open in VS Code
3. Run using Live Server
